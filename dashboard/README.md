# UMKM AI Suite — Dashboard

Dashboard web untuk UMKM AI Suite.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui

## Setup

```bash
cd dashboard
npm install
npm run dev
```

Buka: http://localhost:3000

## Environment Variables

Buat file `.env.local`:

```
NEXT_PUBLIC_BACKEND_URL=http://localhost:8000
```
